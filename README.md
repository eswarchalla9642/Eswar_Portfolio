# Eswar Challa Portfolio

This is a React portfolio website built with Vite.

## Setup

1. Open PowerShell in `c:\Users\srini\Documents\Eswar_portfolio`
2. Run:

```powershell
npm install
```

This installs dependencies locally in the project folder and does not require any global packages.

## Run locally

```powershell
npm run dev
```

Then open the local URL shown in the terminal.

## Build for production

```powershell
npm run build
```

## Preview production build

```powershell
npm run preview
```
